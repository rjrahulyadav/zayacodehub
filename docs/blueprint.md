# **App Name**: Zaya Code Hub: Neon Edge

## Core Features:

- Neon Animated Hero Section: Create a visually striking hero section with neon glow animations and interactive elements to capture user attention.
- Dynamic Portfolio Grid: Develop a filterable portfolio grid showcasing projects with names, descriptions, technologies, and images. Admin has the tool to dynamically update project data and media via the admin dashboard.
- Admin-Controlled Hiring Module: Implement a secure admin dashboard allowing authorized personnel to post, edit, and delete job openings (freshers & interns).
- Interactive Service Cards: Design animated service cards with hover glow effects for each IT service offered (Custom Software, Web, Mobile, IT Consulting, Digital Solutions).
- CEO Showcase: Dedicated section on the About page highlighting CEO Rahul Kumar Yadav with professional photo and title.
- Dark Mode Styled Map: Integrate Google Maps API displaying the company location in Salem, Tamil Nadu, with a dark mode-styled aesthetic.

## Style Guidelines:

- Primary color: Electric Indigo (#6F00FF) to capture the futuristic IT feel.
- Background color: Dark slate gray (#2F4F4F) to contrast against the neon elements and ensure readability.
- Accent color: Cyan (#00FFFF) for interactive elements and highlights, providing a vivid contrast.
- Body and headline font: 'Space Grotesk' sans-serif for headlines, paired with 'Inter' sans-serif for body, creating a computerized modern style.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use glowing neon icons in blue and cyan, aligning with the futuristic theme and representing various IT services.
- Implement a responsive grid layout with glassmorphism cards, ensuring a premium startup-style look and smooth transitions on all devices.
- Use Framer Motion for subtle yet engaging animations, including neon glow effects, hover animations, and smooth scroll transitions.
